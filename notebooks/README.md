# Notebook Guide

You can create these notebooks from the Python modules in this project:

1. `01_data_cleaning_eda.ipynb`
   - Load the dataset
   - Clean missing values
   - Convert `last_updated` to datetime
   - Create temperature and precipitation charts

2. `02_forecasting_models.ipynb`
   - Select one city
   - Create lag and rolling features
   - Train baseline, Random Forest, Gradient Boosting, and ensemble models
   - Compare MAE, RMSE, and R2

3. `03_advanced_analysis.ipynb`
   - Detect anomalies
   - Analyze air quality correlations
   - Show feature importance
   - Map spatial patterns

The Streamlit app already includes the core workflow, so the notebooks can be used mainly for explanation and screenshots.
